# Blinkit Sales Analysis Dashboard 📊

This project is a Power BI dashboard built to analyze Blinkit sales data.

## 🔹 Key Insights:
- Total Sales: 18.59M
- Average Sales: 2.18K
- Item Type Analysis
- Outlet Size Distribution
- Location-wise Sales

## 🔹 Tools Used:
- Power BI
- Excel

## 🔹 Features:
- Interactive Filters (Slicers)
- Category-wise Analysis
- Location-based Insights
